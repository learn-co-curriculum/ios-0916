## Sample problems for Homework


```
"Given an array of numbers, write a method that calculates and returns the sum of all the numbers.
Sample input: [3, 1056, 10, -5, 0, 11.5]
Sample output: 1075.5"
```



```
"Given a positive integer, write a method that returns an array with each individual digit as a separate element.
Sample input: 1987647
Sample output: [1, 9, 8, 7, 6, 4, 7]
```


```
"Given an integer n, print a “staircase” of height n comprised of the character #. The top level should have 1 # while the bottom should have n #’s.
Sample input: 6
Sample output:
#
##
###
####
#####
######
```
